# accessibility-low-load-companion

Use this skill when the user's accessibility bottleneck is mixed or unclear and no narrower accessibility skill clearly owns the first step.

Examples:

```text
Use accessibility-low-load-companion. I have rough notes, typos, and too much to read. Choose the smallest useful next step.
```

```text
Use accessibility-low-load-companion. I am not sure whether this needs dictation cleanup, reading triage, or prose repair.
```

Follow `docs/SKILL_OPERATIONAL_BOUNDARIES.md`. Use `docs/ROUTING_MATRIX.md` for route selection and `docs/SOURCE_LIMITS.md` when facts, privacy, evidence, or external sharing matter.
