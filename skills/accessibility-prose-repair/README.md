# accessibility-prose-repair

Use this skill for meaning-preserving spelling, grammar, punctuation, sentence-boundary, paragraph-break, typo, or local readability repair in existing prose.

Examples:

```text
Use accessibility-prose-repair. Fix spelling and sentence boundaries without changing my meaning.
```

```text
Use accessibility-prose-repair. Clean this email for grammar and readability, but do not add commitments.
```

Follow `docs/SKILL_OPERATIONAL_BOUNDARIES.md`. Use `docs/SOURCE_LIMITS.md` when the prose includes factual claims, citations, private details, external sharing, or high-stakes decisions.
