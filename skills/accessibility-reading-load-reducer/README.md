# accessibility-reading-load-reducer

Use this skill to reduce reading burden from dense documents, long emails, instructions, policies, source packets, reading lists, or long notes before summarizing or deciding.

Examples:

```text
Use accessibility-reading-load-reducer. Tell me what to read closely, skim, park, or skip in this long document.
```

```text
Use accessibility-reading-load-reducer. I have 20 links and only 30 minutes. Build a low-load reading plan.
```

Follow `docs/SKILL_OPERATIONAL_BOUNDARIES.md`. Use `docs/SOURCE_LIMITS.md` when the material includes factual claims, citations, current facts, high-stakes decisions, or external sharing.
