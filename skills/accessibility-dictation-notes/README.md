# accessibility-dictation-notes

Use this skill for dictated thoughts, voice transcripts, speech-to-text output, meeting notes, auto-captions, or spoken fragments that need structure without a prose rewrite.

Examples:

```text
Use accessibility-dictation-notes. Turn this voice note into cleaned notes, open questions, and next actions.
```

```text
Use accessibility-dictation-notes. Segment this meeting transcript into decisions, tasks, ambiguity, and follow-up.
```

Follow `docs/SKILL_OPERATIONAL_BOUNDARIES.md`. Use `docs/SOURCE_LIMITS.md` when transcripts include factual claims, private people, research, external sharing, or high-stakes decisions.
